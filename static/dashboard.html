Orchestrator AI decide fluxul: analizează trenduri, alocă taskuri workerilor, optimizează producția. Pipeline-ul de mai jos este conectat la API-ul live.

Pipeline complet (live endpoints)

Trend Detection ↓ Idea Swarm ↓ Style Cloning ↓ Script Engine ↓ Video Rendering ↓ Virality Model ↓ A/B Testing ↓ Scheduler

1. Agent Orchestrator Layer

1 Agent Orchestrator

AI orchestrator decide ce se întâmplă în pipeline

  • Analizează trenduri
  • Decide ce idei merită generate
  • Alocă taskuri workerilor (Celery)
  • Optimizează producția video

Decizii: câte idei, ce tip video, ce stil creator.

→ API & workers

2. Trend Intelligence Engine

2 Trend Intelligence

Surse: TikTok hashtags, YouTube Shorts, Google Trends, Reddit

  • engagement velocity, share velocity
  • comment growth, creator growth rate
  • Output: trend_score, time_to_viral

POST /trends/update

3. Idea Generation Swarm

3 Idea Swarm

Cluster de AI idea agents — exemplu: 500 trenduri × 100 idei = 50.000 idei

  • hook, concept, caption, hashtags, trend_angle

POST /ideas/generate

4. Creator Style Cloning

4 Style Cloning

Analiză creatori virali TikTok — hook pattern, speech speed, caption structure, video pacing, editing style

  • Output: style embeddings pentru conținut în stilul creatorilor virali

(În roadmap — style embeddings)

5. Script Generation Engine

5 Script Engine

Idei → scripturi: 3s hook, 15–30s narațiune, scene breakdown, text overlays

  • Scene 1 — Hook · Scene 2 — Context · Scene 3 — Reveal · Scene 4 — CTA

POST /scripts/generate

6. Video Rendering Cluster

6 Video Rendering

Script → scene generator → image/video assets → voiceover AI → subtitles → FFmpeg render

  • Output: 9:16 vertical, 1080×1920, 15–30 sec

POST /videos/render

7. Virality Prediction Engine

7 Virality Model

Features: hook strength, trend score, hashtags, topic popularity, caption sentiment

  • Output: virality_score (0–1)
  • Filtru: publish only if score > 0.75

POST /virality/score

8. Auto A/B Testing System

8 A/B Testing

Per video: 3 hooks × 2 captions × 2 hashtag sets = 12 variante. Sistemul testează și învață din performanță.

  • În roadmap — A/B variants & learning

9. Content Scheduler

9 Scheduler

Videouri aprobate programate pentru TikTok, Instagram Reels, YouTube Shorts

  • Optimizare: posting time, hashtags, caption

POST /schedule/create

Fluxuri alternative (live)

Clips (long-form → shorts) · Repo → idea / to-video
